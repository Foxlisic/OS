# Техническая информация о разметке диска в отладочном режиме BOCHS

Для Bochs создается отдельная дискета FDD. В первом сектора содержится простейший загрузочный код, который пишет первые 64Кб кода
с дискеты.

Дискета является виртуальной и не является настоящим источником данных. Предназначено только для тестирования.

FIXED disk: 0x200000 начинается 1-й сектор
