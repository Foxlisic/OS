# Полный сбор: 
# Требуется makefiles с правильной разметкой   

    <timestamp> <file>

# 1) компиляция [c->s], 

    cd %s && gcc -c -masm=intel -m32 -fno-asynchronous-unwind-tables %s -S

# 2) трансляция [s->asm], 
    
    ./fas %s.s

# 3) Ассемблирование, 4) Копирование на virtual fdd

    fasm kernel.asm

Выполнить команду

    ./make

# ----------------------------
# Процесс сборки
# ----------------------------

# Записать загрузочный сектор

    ./bootmake dcopy

# Прокомпилировать готовые коды и записать ядро

    ./bootmake copyk

# fas -- Перекодирование .s файлов в .asm для fasm    
# bootmake -- создание загрузочного диска и ассемблирование
