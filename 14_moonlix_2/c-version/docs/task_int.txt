При загрузке драйвера, будет регистрироваться определенный INT, который
будет обрабатывать данные.

    [!] Регистрация INT в системе доступна только драйверам (1-й RING).

Регистрироваться будет именно задача с общим call-gate, а это именно eip=0.
При вызове прерывания откуда угодно прерывание будет передано задачей с
загрузкой TSS, и оттуда уже обрабатываться.
