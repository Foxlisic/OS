; Прерывания
; ----
0x00000 256 векторов прерываний
0x07000 VESA Information

; Страницы
; ----
0x9A000 Первая PDP-таблица
0x9B000 Первая Page Directory
0x9C000 Каталог страниц [4 страницы] для описания первых 4 Гб памяти

; ---- Постраничник, 4 Гб ---
0x100000 8 Мб страниц, описывающие 4 Гб всей памяти. Тут же LFB

; -----
0x900000 128к элементов каталогов памяти
0xA00000 Неразмеченная память общего назначения
